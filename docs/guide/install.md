# Isntall & Usage

## Install

### Yarn

```
yarn add @webzlodimir/vue-avatar
```

### Npm 

```
npm i @webzlodimir/vue-avatar
```

## Usage

### Vue 3

Just import component where you want

```js
import {VueAvatar} from '@webzlodimir/vue-avatar'
```

And use it in `template` tag

```vue
<template>
  <vue-avatar username="Simple User" />
</template>
```
